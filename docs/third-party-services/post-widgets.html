Widgetpack Rating

Rating service with login through any social networks like Facebook, Twitter, Google+, LinkedIn, Instagram and others.

  • Get ID →
  • NexT Config

Login to widgetpack and add new site, you will get your site ID:

You can enable it by setting value rating.enable to true, and copy ID value of id in theme config file. You can also configure the rating color by editing the value of rating.color.
